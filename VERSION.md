# 2026 世界杯实时数据页面 — 版本记录

## v2-final (2026-06-20)

**存档文件**：
- `index-v2-final.html` — 页面快照
- `data/worldcup-live-v2-final.json` — 数据快照

**版本特性**：

### 四视图
1. **今日焦点** — 近期完赛 + 即将开赛，表格化排版（与积分榜风格统一）
2. **积分榜** — 12组小组积分，中文队名
3. **全部赛程** — 水平日期时间轴，按日期过滤
4. **淘汰赛对阵** — 经典树状图（上半区-决赛-下半区），无滑条

### 淘汰赛版面
- 参考 2018 世界杯对阵图板式，左右对称树形
- 上半区（32强→16强→8强→半决赛）| 决赛+季军赛 | 下半区
- 决赛卡片金色高亮，队名15px/比分18px 突出
- 每场显示 ⏰北京时间 + 📍中文场馆

### 中文全覆盖
- 48 支球队 → TEAM_CN 映射（build_all.py 嵌入）
- 16 个场馆 → STADIUM_CN 映射
- 界面全中文

### 数据
- 来源：MCP `wc__*` 工具实时拉取
- `build_all.py` 处理：北京时间计算、中文映射、数据合并
- 输出 `worldcup-live.json` 供前端 fetch

### 部署
- `python -m http.server 8766 --bind 127.0.0.1` 在 `worldcup/` 目录
- 自动化任务：每小时刷新数据
